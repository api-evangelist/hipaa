# HIPAA (hipaa)
HIPAA (Health Insurance Portability and Accountability Act) is U.S. legislation providing data privacy and security provisions for safeguarding medical information. HIPAA compliance is required for healthcare providers, health plans, and their business associates handling protected health information (PHI).
